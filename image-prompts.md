## How to assemble the final post

1. Open `carousel.html` in Chrome at 100% zoom.
2. For each slide screenshot at exactly 1080×1350 (Instagram portrait).
3. Replace the CSS background panels with the generated photo for each slide as a `background-image` (keep the existing dark vignette so the text stays legible).
4. Export each slide as JPG (sRGB, quality 90+) for upload.
